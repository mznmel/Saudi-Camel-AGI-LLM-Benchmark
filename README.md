# Saudi Camel AGI LLM Benchmark 🐪

Inspired by [@simonw's "pelican bicycle"](https://github.com/simonw/pelican-bicycle) benchmark, this benchmark tests how well LLMs can generate an SVG of a Saudi camel building an AGI.

## Why This Benchmark?

This test evaluates three aspects of LLM capabilities:
- How LLMs depict camels
- Understanding of Saudi cultural references
- How they imagine AGI systems

All in one simple visual test.

## Prompt

```
Generate an SVG illustration of a Saudi camel building an artificial general intelligence system.
```
